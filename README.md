# 🛠️ Prerequisites

Before running this project, ensure you have the following installed:

## 1️⃣ Install Node.js (Required)

Download and install **Node.js (LTS version recommended)**:

🔗 https://nodejs.org/

After installing, verify the installation by running:

``` bash
node -v
npm -v
```

If both commands return version numbers, you're good to go ✅

------------------------------------------------------------------------

# 📥 Step-by-Step Setup Guide

## 1️⃣ Clone the Repository

``` bash
git clone <YOUR_GITHUB_REPO_URL>
```

### Example:

``` bash
git clone https://github.com/Achinthya630/MarkeTeer.git
```

------------------------------------------------------------------------

## 2️⃣ Navigate Into the Project Folder

``` bash
cd marketing_tool
```

------------------------------------------------------------------------

## 3️⃣ Install Dependencies

This installs all required packages listed in `package.json`.

``` bash
npm install
```

This may take a few seconds ⏳

------------------------------------------------------------------------

## 4️⃣ Run the Development Server

``` bash
npm run dev
```

After running the command, you should see something like:

    Local: http://localhost:5173/

Open that link in your browser 🌐

Your application should now be running successfully 🚀
